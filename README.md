## 🚀 طريقة التشغيل

### ▶️ محلياً — بدون خادم

```bash
# 1. استنساخ المستودع
git clone https://github.com/Ahmedtaha2008/My-web.git

# 2. الدخول للمجلد
cd My-web

# 3. فتح index.html في المتصفح مباشرة
# (Windows)
start index.html

# (macOS)
open index.html

# (Linux)
xdg-open index.html
```

### 🌐 محلياً — مع خادم مصغّر (مستحسن)

```bash
# باستخدام Python
python -m http.server 8000

# باستخدام Node.js
npx serve .

# باستخدام VS Code
# → تثبيت إضافة "Live Server" والضغط على "Go Live"
```

ثم افتح: **`http://localhost:8000`**

### ☁️ النشر على GitHub Pages (مجاني)

```
1. اذهب إلى Settings في المستودع
2. اختر "Pages" من القائمة الجانبية
3. Source → Deploy from a branch
4. Branch → main / (root)
5. احفظ → سيصبح الموقع متاحاً خلال دقائق
```

**الرابط:** `https://ahmedtaha2008.github.io/My-web/`

### 🔧 خيارات نشر أخرى

| المنصة | الطريقة | الوقت |
|--------|---------|-------|
| **Netlify** | اسحب المجلد وأفلته في [app.netlify.com](https://app.netlify.com) | < 1 دقيقة |
| **Vercel** | `npx vercel --prod` أو ربط GitHub | < 2 دقيقة |
| **Cloudflare Pages** | ربط المستودع من لوحة التحكم | < 3 دقائق |

## 🎨 التخصيص

### تغيير الألوان الأساسية

افتح `style.css` وعدّل متغيرات `:root`:

```css
:root {
  --primary: #6c5ce7;     /* اللون البنفسجي الأساسي */
  --secondary: #00cec9;   /* اللون الفيروزي الثانوي */
  --accent: #fd79a8;      /* لون التمييز الوردي */
  --gradient: linear-gradient(135deg, var(--primary), var(--secondary));
}
```

### تعديل المحتوى الشخصي

```
index.html
├── السطر ~55     → تعديل الاسم والوصف في Hero Section
├── السطر ~120    → تعديل معلومات "عني"
├── السطر ~170    → تعديل المهارات والنسب المئوية
├── السطر ~220    → إضافة / حذف مشاريع Portfolio
└── السطر ~300    → تعديل معلومات التواصل
```
